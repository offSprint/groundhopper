# ⚽ Groundhopper - Футбольный дневник

Telegram Mini App для граундхопперов — отслеживание посещённых матчей, стадионов и клубов.

## Возможности
- 📊 Статистика матчей, стран, стадионов
- 🗺️ Карта посещённых стадионов
- 🌐 Поддержка 8 языков
- 👑 Админ-панель

## Установка
1. Скопируйте файлы на сервер
2. Импортируйте install.sql в базу данных
3. Настройте api.php
4. Откройте index.html
